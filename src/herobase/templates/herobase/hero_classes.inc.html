{% blocktrans %}Die Heldenklasse Deines Charakters gibt an welche Arten von Quests Du bevorzugst. Natürlich kannst Du jede Quest annehmen, die Dir gefällt, aber Deine Klasse sollte eine gewisse Präverenz zum Ausdruck bringen.{% endblocktrans %}

{% blocktrans %}Jede Quest ist entsprechend einer dieser Heldenklassen zugeordnet.{% endblocktrans %}

{% trans "Action" %}

{% blocktrans %}Der Actionheld will raus und die Welt verändern. Wenn er die Dinge nicht in die Hand nimmt, wer dann? Das aktive VERÄNDERN eines für ihn inakzeptablen Zustands ist sein Motto.{% endblocktrans %}

{% blocktrans %}Actionquests können z.B. die Aufforderung des Besuchens und/oder organisieren von Demos oder anderen Veranstaltungen sein.{% endblocktrans %}

{% trans "Protective" %}

{% blocktrans %}Der Protective Hero ist besorgt um Dinge und Wesen. Zu BESCHÜTZEN und zu Bewahren ist sein Ziel{% endblocktrans %}

{% blocktrans %}Protective Quests können Tier oder dritte-Welt-Patenschaften sein, Blutspenden, mit Denkmal- und Umweltschutz zusammenhängende Quests, usw.{% endblocktrans %}

{% trans "Diplomat" %}

{% blocktrans %}Menschen und Nationen miteinander zu VERBINDEN ist die Mission des Diplomaten; eine Mission, die sich manchmal als wichtiger erweisen mag, als alle anderen zusammen.{% endblocktrans %}

{% blocktrans %}Diplomatenquests können das Mitsingen in einem Chor sein, Übersetzungen von Texten, Anbieten eines Couchsurfingplatzes usw.{% endblocktrans %}

{% trans "Gadgeteer" %}

{% blocktrans %}Wie sieht die Welt aus, in der wir uns bewegen? Wer gestaltet sie, wenn nicht der Gadgeteer? Kreation und KREATIVITÄT sind sowohl die Mittel als auch das Ziel.{% endblocktrans %}

{% blocktrans %}Gadgeteerquests können unter Anderem das Erstellen von Designartikeln aus Müll sein, das Erzeugen von Altpapier, oder das Bauen einer Baumhütte, etc.{% endblocktrans %}

{% trans "Scientist" %}

{% blocktrans %}Wissenschaft, Entwicklung und Wissen - dass unsere Gesellschaft da ist, wo sie jetzt ist, ist allen frei denkenden Geistern zu verdanken, die je auf diesem Planeten gelebt haben. Diese Werte gehören allen und es ist das Ziel des Scientist nicht nur die Grenzen des Wissens auszuloten und zu erweitern, sondern den Zugang für jedermann zu gewährleisten und zu beschützen.{% endblocktrans %}

{% blocktrans %}Scientistquests können alle Arten von Open Source Projekten sein, Umfragen, Einträge in die Wikipedia, usw.{% endblocktrans %}
